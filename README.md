# 💻 PC STORE — Premium PC Builder & Live E-Commerce Platform

Высокотехнологичный интернет-магазин геймерских компьютеров, премиальных комплектующих и периферии. Проект разработан на базе современного веб-фреймворка **Django 6.0**, расширен асинхронным движком **Django Channels (WebSockets)** для административной панели реального времени и снабжен интерактивным фронтендом премиум-уровня.

Приложение полностью готово к сдаче и защите, строго соответствуя всем критериям технического задания и методических рекомендаций (включая CRUD, связи ManyToMany, кастомные роли, сессии, AJAX-запросы, юнит-тестирование и геймификацию).

## 🚀 Ключевые технологические особенности и фичи

### 1. Архитектура базы данных и Django ORM
* **Связность моделей:** Архитектура включает 10 взаимосвязанных моделей (`Product`, `Category`, `Brand`, `Tag`, `Cart`, `Order`, `OrderItem`, `Favorite`, `Review`, `UserProfile`).
* **Связи ManyToMany:** Реализована обязательная по ТЗ связь **Многие-ко-Многим** между товарами и тегами.
* **Бизнес-логика в ORM:** Автоматический расчет средних рейтингов товаров на основе отзывов клиентов, динамический пересчет остатков на складах и агрегация финансовых показателей для админки.

### 2. Асинхронный Live-Дашборд (WebSockets)
* В проект интегрирован полноценный стек асинхронного веб-взаимодействия через **ASGI** сервер и **Django Channels**.
* Класс `DashboardConsumer` обрабатывает постоянное WebSocket-соединение.
* При оформлении заказа клиентом бэкенд мгновенно отправляет событие на фронтенд. В кастомной админ-панели новые заказы всплывают **в режиме реального времени** со звуковым уведомлением без перезагрузки страницы.

### 3. Интерактивный Конфигуратор ПК (`builder.html`)
* Пошаговый мастер подбора комплектующих.
* **Калькулятор TDP:** Алгоритм JavaScript на лету суммирует энергопотребление всех выбранных компонентов. При нехватке мощности блока питания выводится предупреждение.
* **FPS Predictor:** Динамический расчет ожидаемой производительности сборки (средний FPS) в современных играх.

### 4. Геймификация и Программа лояльности
* Модель `UserProfile` связана с системным пользователем связью `OneToOne` и содержит баланс виртуальной валюты **PC Coins** (начисляется как кэшбек 5%).
* **Колесо Фортуны на HTML5 Canvas:** В профиле реализована интерактивная рулетка. Пользователь может крутить колесо, тратя PC Coins, и выигрывать реальные промокоды на скидку.

### 5. Продвинутый UI/UX и AJAX Фронтенд
* **3D-Hologram Эффект:** Карточки товаров отслеживают координаты мыши, создавая иллюзию объемной голограммы.
* **Без перезагрузок (Fetch API):** Добавление в корзину, изменение количества товаров, лайки в избранное происходят асинхронно.
* Поддерживается полноценная **Темная и Светлая темы** с сохранением состояния в `localStorage`.

## 🛠 Технологический стек

* **Backend:** Python 3.10+, Django 6.0.6, Django Channels 4.3.2.
* **Frontend:** HTML5, CSS3, JavaScript (ES6 Vanilla, Fetch API, Canvas API), Bootstrap 5.3, TailwindCSS (в админке), Chart.js.
* **База данных:** SQLite3 (реляционная файловая СУБД, идеальная для локальной демонстрации).

## ⚙️ Инструкция по локальному развертыванию (ЗАПУСК ПРОЕКТА)

Выполните следующие шаги для запуска проекта на вашем компьютере:

### 1. Подготовка папки и окружения
Открой терминал (командную строку) и перейди в корневую директорию проекта (туда, где лежит файл `manage.py`):
```bash
cd путь_к_папке_с_проектом
```

### 2. Создание и активация виртуального окружения (venv)
* **Для Windows:**
```bash
python -m venv venv
venv\Scripts\activate
```
* **Для macOS / Linux:**
```bash
python3 -m venv venv
source venv/bin/activate
```
*(После активации слева в терминале появится надпись `(venv)`)*

### 3. Установка всех зависимостей
Установи библиотеки, необходимые для работы проекта:
```bash
pip install --upgrade pip
pip install -r requirements.txt
```

### 4. Создание Базы Данных (Миграции)
Эта команда автоматически создаст файл базы данных `db.sqlite3` и все нужные таблицы:
```bash
python manage.py migrate
```

### 5. Создание Администратора (Суперпользователя)
Это нужно для входа в админ-панель:
```bash
python manage.py createsuperuser
```
*(Система попросит ввести логин, email и пароль. Пароль при вводе не отображается — это нормально, просто введите его и нажмите Enter)*

### 6. Запуск локального сервера
Запусти проект:
```bash
python manage.py runserver
```

### 7. Проверка в браузере
Открой любой браузер и перейди по ссылкам:
* **Витрина магазина:** http://127.0.0.1:8000/
* **Кастомный Live-Дашборд админа:** http://127.0.0.1:8000/admin-dashboard/
* **Стандартная панель Django:** http://127.0.0.1:8000/admin/
